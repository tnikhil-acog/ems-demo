## Summary: Manager Pages

| Page | Purpose | Key Actions |
|------|---------|-------------|
| **Dashboard** | Overview of team & projects | View team, View projects, See alerts |
| **My Team** | Manage direct reports | View profiles, Assign to projects, Check availability |
| **My Projects** | Manage all projects you lead | View details, Manage teams, Check health |
| **Project Detail** | Deep dive into one project | Edit team, Change allocations, View reports |
| **Find Resources** | Search & assign employees | Search by skills, Assign to project, Request transfers |
| **Requests** | Handle transfer requests | Approve/reject incoming, Track outgoing |

---

## Design Principles

1. **Clean & Minimal:** No clutter, focus on key info
2. **Action-Oriented:** Every screen has clear next steps
3. **Data-Dense:** Show important metrics without overwhelming
4. **Color-Coded Status:** Green (good), Yellow (warning), Red (critical)
5. **One-Click Actions:** Assign, View, Approve - all one click away
6. **Responsive:** Works on desktop, tablet (mobile optional)

---

## Color Scheme

- **Primary:** Blue (#3B82F6) - Actions, links
- **Success:** Green (#10B981) - Healthy, available, positive
- **Warning:** Yellow/Orange (#F59E0B) - At risk, conditional
- **Danger:** Red (#EF4444) - Blocked, critical, urgent
- **Neutral:** Gray (#6B7280) - Inactive, secondary info
- **Background:** White (#FFFFFF) / Light Gray (#F9FAFB)
